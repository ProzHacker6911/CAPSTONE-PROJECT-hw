Learn HTML, CSS, JavaScript, and Bootstrap – Full Beginner Guide

Introduction
If you've ever wanted to create a website, you've probably heard terms like HTML, CSS, JavaScript, and Bootstrap. These are the core technologies used in front-end web development. In this guide, you'll learn what each of them does, how they work together, and how you can use them to build your own webpages from scratch.

What Is HTML?
HTML stands for HyperText Markup Language. It is the language used to create the structure of webpages.

Think of HTML as the skeleton of a website. It tells the browser what content should appear and in what order. HTML uses "tags" to define elements like headings, paragraphs, images, links, tables, and forms.

Example of HTML:
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>This is a simple webpage.</p>
    <a href="https://example.com">Visit Example</a>
  </body>
</html>

HTML Tags:
<h1> to <h6> – Headings  
<p> – Paragraph  
<a> – Link  
<img> – Image  
<ul>, <ol>, <li> – Lists  
<div> – Container (generic block)  
<span> – Inline text container  
<form> – User input forms

HTML is the first thing you should learn when building websites.

What Is CSS?
CSS stands for Cascading Style Sheets. It is used to style HTML elements – in other words, it makes the website look nice.

CSS can change colors, fonts, sizes, spacing, layout, and more. Without CSS, websites would just look like plain text.

Example of CSS:
body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}

h1 {
  color: navy;
  text-align: center;
}

p {
  color: #333;
}

How to Add CSS:
Inline:  
<p style="color: red;">This is red text.</p>

Internal:  
<style>
  p { color: blue; }
</style>

External:  
<link rel="stylesheet" href="styles.css">

CSS Selectors:
* – All elements  
p – Paragraphs  
.classname – Class  
#idname – ID  
div > p – Child element

What Is JavaScript?
JavaScript is a programming language that makes your website interactive.

With JavaScript, you can respond to user actions like clicking buttons, filling forms, scrolling the page, or typing in text boxes. JavaScript is also used for animations, dropdowns, sliders, and more.

Example of JavaScript:
<button onclick="sayHello()">Click Me</button>

<script>
  function sayHello() {
    alert("Hello, world!");
  }
</script>

What JavaScript Can Do:
- Show alerts and messages  
- Change content on the page  
- Show or hide elements  
- Validate form input  
- Fetch data from servers (using fetch or AJAX)

JavaScript Placement:
Inline: <button onclick="myFunction()">Click</button>  
Script tags at bottom: <script>...</script>  
External file: <script src="script.js"></script>

What Is Bootstrap?
Bootstrap is a front-end framework that helps you build modern, responsive websites faster.

It includes:  
- A mobile-first grid system  
- Pre-built components (buttons, modals, forms, navbars)  
- Built-in CSS styles and JavaScript plugins

How to Add Bootstrap:
CDN:  
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">  
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

Bootstrap Button Example:
<button class="btn btn-primary">Click Me</button>

Bootstrap Grid:
<div class="container">
  <div class="row">
    <div class="col-md-6">Left Side</div>
    <div class="col-md-6">Right Side</div>
  </div>
</div>

How They Work Together:
HTML - Structure  
CSS - Styling  
JavaScript - Interactivity  
Bootstrap - Prebuilt UI and responsiveness

Dark Mode Toggle Example:
<button onclick="toggleDarkMode()">Toggle Dark Mode</button>

<script>
function toggleDarkMode() {
  document.body.classList.toggle("dark-mode");
}
</script>

<style>
.dark-mode {
  background-color: #333;
  color: white;
}
</style>

Best Practices:
- Use semantic HTML tags  
- Use separate CSS and JS files  
- Use Bootstrap classes to save time  
- Test on multiple devices  
- Use browser developer tools for debugging

Summary Table:
Technology    | Purpose            | File Extension  
--------------|--------------------|----------------
HTML          | Structure          | .html  
CSS           | Styling            | .css  
JavaScript    | Interactivity      | .js  
Bootstrap     | UI Framework       | (CDN or local)

Conclusion:
Now you know the basics of front-end development using HTML, CSS, JavaScript, and Bootstrap. Practice regularly and build small projects. Explore W3Schools, MDN, and CodePen to sharpen your skills.
