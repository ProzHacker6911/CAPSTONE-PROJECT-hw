HTML

HTML (HyperText Markup Language)
HTML is the foundation of any webpage. It defines the structure and content of the page using elements called tags. These tags tell the browser how to display text, images, links, and other content. For example, you use <h1> for a main heading, <p> for paragraphs, and <img> to show images. Without HTML, websites wouldn’t have any meaningful content to display.











CSS

CSS (Cascading Style Sheets)
CSS controls the style and layout of a webpage. It lets you change colors, fonts, spacing, and positions of HTML elements to make the page look attractive and organized. For example, CSS can make all paragraphs blue, set the background color, or create responsive layouts that adjust on different screen sizes. By separating content (HTML) from style (CSS), web design becomes more flexible and easier to maintain.











JS

JavaScript (JS)
JavaScript is the programming language that makes web pages interactive. With JavaScript, you can respond to user actions like clicks, form submissions, or keyboard input. It can dynamically update content, validate forms, create animations, and communicate with servers without reloading the page. JS turns static HTML and CSS into lively, engaging experiences.











BOOTSTRAP

Bootstrap
Bootstrap is a popular front-end framework that helps you build responsive, mobile-first websites quickly. It provides pre-made CSS styles and JavaScript components, like buttons, navbars, modals, and grids, so you don’t have to write everything from scratch. Bootstrap’s grid system also helps in designing layouts that automatically adjust for desktops, tablets, and phones.
